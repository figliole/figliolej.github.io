Software Testing
                                
                                    WebDriver driver = new >ChromeDriver(); 
                                    driver.get("il link del sito da aprire"); //Apre la pagina nel browser indicato in precedenza

                                    WebElement emailInput = driver.findElement(By.id("//inputEmail")); //Trova l'input tramite l'id "inputEmail"
                                    emailInput.sendKeys("figlioliacat@gmail.com"); //Inserisce il valore "figlioliacat@gmail.com"
                            
                                    WebElement passwordInput = driver.findElement(By.id("//inputPassword")); //Trova l'input tramite l'id "inputPassword"
                                    passwordInput.sendKeys("Aldo1234!"); //Inserisce il valore "Aldo1234!"

                                    WebElement button = driver.findElement(By.id("//buttonConferma")); //Trova il button tramite l'id "buttonConferma"
                                    button.click(); //Effettua il click al button

                                     Thread.sleep(2000); //Blocca il test per 2 sec
Questo è uno script, scritto in Java utilizzando il framework Selenium, che permette di compilare automaticamente i campi di input e, successivamente, cliccare sul button di conferma. Selenium è uno strumento per automatizzare l'interazione con le pagine web e per eseguire test in modo efficiente. E' molto flessibile e questo permette di integrarsi con diversi linguaggi di programmazione lo rendono una scelta popolare per gli sviluppatori e i tester
Ho avuto anche la possibilità di toccare con mano tutta la fase relativa al testing.
Dallo stilare la documentazione alla parte relativa al Test Automation, passando dalla creazione (tramite Jira) di casi di test e, svolgendo test funzionali e non, di bug.
